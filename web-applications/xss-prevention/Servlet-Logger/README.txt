Servlet-Logger
---------------------------------------------------------------------

This Web application is used to log request parameter names and values.

After deployment, we can send data using the following URL:

http://localhost:8080/Servlet-Logger/controller?id=7&name=homer

Which results in the following console output:

Parameters from 127.0.0.1 at Thu May 29 20:55:06 CEST 2014:
    id  = 7
    name  = homer
