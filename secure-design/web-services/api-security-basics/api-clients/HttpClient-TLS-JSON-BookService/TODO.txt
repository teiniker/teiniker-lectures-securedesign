xercise: HttpClient + TLS + JSON Mapping (BookService)
-------------------------------------------------------------------------------

Implement test cases for the given BookService, including TLS connection and
JSON mapping.

Setup:

$ cd api-tls/SpringBoot-BookService-TLS 
$ mvn spring-boot:run 

$ curl -ik https://localhost:8443/books

A) Data Transfer Object (DTO)
    Implement a DTO named Book containing the following properties:
    - id:long
    - author:String
    - title:String 
    - isbn:String 

B) Replace the following curl statements by test cases:
    testFindAll():  curl -i http://localhost:8080/books
    testFindById(): curl -i http://localhost:8080/books/2
    
    testFindByInvalidId(): curl -i http://localhost:8080/books/666
    testInsert():   curl -i -X POST http://localhost:8080/books -H 'Content-type:application/json' -d '{"author":"Robert C. Martin","title":"Clean Code","isbn":"978-0132350884"}'
    testUpdate():   curl -i -X PUT http://localhost:8080/books/1 -H 'Content-type:application/json' -d '{"author":"Joshua Bloch","title":"Effective Java, 2nd Edition","isbn":"978-0134685991"}'
