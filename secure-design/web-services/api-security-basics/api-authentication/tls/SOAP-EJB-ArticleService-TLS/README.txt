How to access the WSDL definition?
-------------------------------------------------------------------------------

URL: http://localhost:8080/SOAP-EJB-ArticleService/ArticleService?wsdl

$ curl -i -X GET http://localhost:8080/SOAP-EJB-ArticleService-TLS/ArticleService?wsdl
HTTP/1.1 302 Found
Connection: keep-alive
Location: https://localhost:8443/SOAP-EJB-ArticleService-TLS/ArticleService?wsdl
Content-Length: 0
Date: Thu, 29 Oct 2020 09:16:16 GMT

$ curl -i -k -X GET https://localhost:8443/SOAP-EJB-ArticleService-TLS/ArticleService?wsdl

Note that the port binding in the WSDL is not correct:
    <soap:address location="http://localhost:8080/SOAP-EJB-ArticleService-TLS/ArticleService"/>


findAll()
-------------------------------------------------------------------------------
<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:ser="http://service.lab.se.org/">
   <soapenv:Header/>
   <soapenv:Body>
      <ser:findAll/>
   </soapenv:Body>
</soapenv:Envelope>

<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
   <soap:Body>
      <ns2:findAllResponse xmlns:ns2="http://service.lab.se.org/">
         <return id="1" price="4295">
            <description>Design Patterns</description>
         </return>
         <return id="2" price="3336">
            <description>Effective Java (2nd Edition)</description>
         </return>
      </ns2:findAllResponse>
   </soap:Body>
</soap:Envelope>
